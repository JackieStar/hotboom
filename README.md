# hotboom

### 项目技术栈
koa2 + mysql + es6 
### 编程方式
面向对象编程 class

### 项目启动
```js
npm install

npm run start:dev
```
### mySql配置
```js
 database:{
        dbName:'xxx', // 修改自己的数据库名
        host:'localhost',
        port:3306,
        user:'root',
        password:'123456'
    }
```
